Custom Web Stack Template
================================================

Template for whenever I'm building a new web 
app/toy/experiment/thing to be hosted on GitHub 
using my custom tech stack.

Technology
-----------------------------------------------

| For        | Name           |
|:-----------|:---------------|
| Compiling  | Webpack        |
| Developing | Webpack        |
| Testing    | Cypress        |
| JavaScript | Babel          |
| CSS        | SCSS           |
| Hosting    | GitHub Pages   |
| Automation | GitHub Actions |
